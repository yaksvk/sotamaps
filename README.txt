This is a meta project demonstrating the functionality of Flask
to work as middleware for several applications.

In this case we are using the DispatcherMiddleware which allows
us to serve 2 different applications in two different /urls.

VHF App

This is the source code for the web application at:
https://sotamaps.herokuapp.com/vhf/
To enable amateur radio operators taking part in the VHF Activity
to upload their logs in the ADIF format and get a nice map with QSOs
and result scores.
